## Reporting a Vulnerability

### Contact Information
**Security Team**: security@africacryptochainx.com  
**Emergency Contact**: security-emergency@africacryptochainx.com  
**PGP Key**: Available upon request

### Response Protocol
- **Initial Response**: Within 24 hours of report
- **Status Updates**: Every 48 hours until resolution
- **Critical Vulnerabilities**: Patch within 72 hours
- **Public Disclosure**: Coordinated with reporter after fix deployment

### Vulnerability Types
We welcome reports for:
- Smart contract vulnerabilities
- Cryptographic implementation flaws
- Network security issues
- Authentication/authorization bypasses
- Data privacy concerns
- API security weaknesses

### What to Include
- Detailed description of the vulnerability
- Steps to reproduce
- Potential impact assessment
- Suggested fixes (if any)
- Your contact information

### Security Updates
- Regular security patches released monthly
- Critical updates deployed immediately
- Security advisories published on our website
- CVE assignments for significant vulnerabilities
